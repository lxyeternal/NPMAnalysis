Tool Limitations:

  • Over-reliance on keyword matching without semantic understanding
  • Cannot distinguish between code comments, documentation, and executable code
  • Low confidence detection still generates alerts, creating noise
  • Doesn't consider programming language context (JavaScript vs Shell scripts)
