beautiful-text
==========

Print colorful text in console

## Installation
npm
```bash
npm install beautiful-text
```

component
```bash
component install shallker/beautiful-text
```

## Quick Start
```javascript
var colorConsole = require('beautiful-text');

colorConsole.red('you should see red text');
colorConsole.green('you should see green text');
```

## API
### colorConsole
#### .black(String text)
#### .red(String text)
#### .green(String text)
#### .yellow(String text)
#### .blue(String text)
#### .magenta(String text)
#### .cyan(String text)
#### .white(String text)
#### .grey(String text)
